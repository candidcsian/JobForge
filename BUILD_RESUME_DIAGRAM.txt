                    RESUME GENERATION ALGORITHM

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│  Input: Master Profile + Job Description                               │
│                                                                          │
│  Step 1: Analyze Job                                                   │
│    ├─ Extract required skills                                          │
│    ├─ Identify key responsibilities                                    │
│    └─ Determine experience level                                       │
│                                                                          │
│  Step 2: Match Profile                                                 │
│    ├─ Find matching skills (40 points)                                │
│    ├─ Match job titles (40 points)                                    │
│    └─ Match experience level (20 points)                              │
│                                                                          │
│  Step 3: Generate Resume                                               │
│    ├─ Summary: Highlight relevant experience                          │
│    ├─ Experience: Emphasize matching achievements                     │
│    ├─ Skills: Categorize by relevance                                 │
│    └─ "Why [Company]": Custom section                                 │
│                                                                          │
│  Output: Tailored Resume (Markdown/PDF)                               │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘

                    COMMANDS TO BUILD RESUME

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│  # Option 1: Interactive (Easiest)                                     │
│  python3 interactive.py                                                │

│    → OR enter experience manually                                      │
│    → Saves to career/*.md                                              │

│  # Option 2: Manual Edit                                               │

│                                                                          │
│  # Option 3: Copy existing                                             │

│  python3 jobforge.py match --resume my-resume.pdf                     │
│    → Parses and extracts information                                   │
