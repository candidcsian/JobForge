## How It Works

### Step 1: Run the Agent
```bash
python3 jobforge_agent.py
```

### Step 2: Answer Questions
The agent will ask you about:
- Basic information (name, contact, location)
- Career timeline (when you started)
- Work history (companies, roles, projects)
- Documents you have (resume, work summaries)

### Step 3: Provide Documents (Optional)
You can provide:
- Old resume (PDF/DOCX)
- Work summaries or performance reviews
- LinkedIn profile export
- Or just answer questions manually

### Step 4: Fill in Gaps
The agent identifies missing years and asks specific questions about those periods.

### Step 5: Get Your Files
The agent creates:
- Master resume
- ATS-optimized resume (Word)
- LinkedIn profile content
- Skills matrix
- Job search strategy

### Step 6: Upload & Apply
Follow the agent's instructions to:
- Upload resume to job portals
- Update LinkedIn profile
- Start applying to jobs

## Frequently Asked Questions

### Q: Do I need to provide all my career history at once?
**A**: No! The agent saves your progress. You can provide information in multiple sessions.

### Q: What if I don't have old resumes or documents?
**A**: No problem! The agent will guide you through manual entry with specific questions.

### Q: Can I customize the resume for different roles?
**A**: Yes! The master resume can generate multiple variants (SDET, QA Lead, Mobile QA, etc.)

### Q: Is my data secure?
**A**: Yes! All data stays on your local machine. Nothing is sent to external servers.

### Q: Can I share this tool with friends?
**A**: Absolutely! Each user's data is stored separately. Just share the repository.

### Q: What if I'm currently employed and want to search discreetly?
**A**: The agent provides LinkedIn privacy settings to search without alerting your current employer.

### Q: How long does it take to get results?
**A**: Most users see recruiter calls within 1 week and interview requests within 2-3 weeks.
